## 开发原则

1. **每个任务开始前**: 调用 `get_design_context` 获取该 frame 的完整设计数据
2. **素材提取**: 通过 Figma MCP asset URL 下载原始素材, 不使用截图
3. **颜色/字体/间距**: 必须与 Figma 标注的精确值一致, 不允许"差不多"
4. **完成每个 Phase 后**: 在模拟器截图并与 Figma 设计稿逐一对比
5. **发现偏差立即修正**: 不带着偏差进入下一个 Phase
